 index.html, style.css, .js, .php,...

- Will list files in the director.
	$ ls

- Check the Git status and see it is a part of repo?
	$ git status

  Files in your Git repository folder can be in one of 2 states	
	+ Tracked : các files mà Git biết và được thêm vào repo
	+ Untracked: các files nằm trong thư mục làm việc của bạn
			nhưng không được thêm vào repository.

---Merge-------
Merge the current branch with the branch master, on origin:
	$ git merge origin/master
Merge the branch-1 branch with the current branch:
	$ git merge branch-1
